1. git fetch

git
 fetch հրամանի դեպքում git - ը նշված branch ից բոլոր commit -ները տեղափոխում է ընթացիկ branch
 որի վրա էտ պահին աշխատուն ենք,
և save  է անում local repo. ի վրա



2. git reset --hard origin/master


այս հրամանով git ին ասում ենք ներկա պահի աշխատանքային վիճակը բեր

origin/master վիճակին,
 իսկ --hard -ը նշանակում է որ ignor անի ցանկացած վտանգավոր փոփոխություն


3. git push -u

git push
 -u արգումենտով կիրառելու եպքում  մեր local branch ը
 ավտոմատ կապվում է,
և հետագայում կարող ենք push ը օգտագործել առանց որևէ արգումնետի
